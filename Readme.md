This project is a simple ASP.NET Core application that uses Blazor for the frontend. It demonstrates how to create a web application with a modern user interface using Blazor components.

## Features
- Blazor Server-side rendering
- Responsive design
- Simple navigation
- Integration with ASP.NET Core backend
